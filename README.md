# Merge Function

This project implements a `merge` function that combines three sorted arrays 
(`collection_1`, `collection_2`, and `collection_3`) into a single ascending array.
  
- `collection_1` and `collection_2` are sorted ascending.
- `collection_3` is sorted descending.

---


### Install dependencies
```bash
npm install
```
### Run Test
```bash
npm test
```
